# 🏥 Patient Health Insights Dashboard - Excel-Based Healthcare Analytics

This project presents a highly interactive **Patient Health Insights Dashboard** developed using **Microsoft Excel**. The dashboard leverages Pivot Tables, Pivot Charts, KPI Cards, and Slicers to provide meaningful insights into patient records, hospital performance, medical conditions, and billing data. It is designed to help healthcare administrators and analysts monitor key healthcare metrics and make data-driven decisions.

## 🎯 Dashboard Objectives

The primary objective of this dashboard is to provide comprehensive healthcare insights by analyzing:

* Total number of patients in the dataset.
* Overall billing revenue generated.
* Highest billing amount recorded.
* Most common disease among patients.
* Admission type distribution.
* Revenue contribution by insurance providers.
* Top-performing hospitals by revenue.
* Patient distribution across medical conditions.
* Impact of filters on KPIs and visualizations.

---

## 🚀 Key Features

### 1. KPI Summary Cards

Dynamic KPI cards display important healthcare metrics:

* **Total Patients:** 55,500
* **Highest Billing Amount:** ₹25,539.32
* **Total Billing Revenue:** ₹1,41,74,32,043.40
* **Most Common Disease:** Arthritis

These KPIs automatically update based on selected filters.

---

### 2. Interactive Slicers

Users can dynamically filter the dashboard using:

* Admission Type

  * Elective
  * Emergency
  * Urgent

Additional slicers can be connected to Pivot Tables for deeper analysis.

---

### 3. Dynamic Data Exploration

The dashboard updates all charts and KPI cards in real time when filters are applied, enabling quick and efficient healthcare analysis.

## 📁 Dataset Overview

The dashboard is built using a healthcare dataset containing patient and hospital information.

| Field Name         | Description                        |
| ------------------ | ---------------------------------- |
| Patient ID         | Unique identifier for each patient |
| Patient Name       | Name of the patient                |
| Age                | Patient age                        |
| Gender             | Male/Female                        |
| Admission Type     | Elective, Emergency, or Urgent     |
| Hospital Name      | Name of the hospital               |
| Insurance Provider | Insurance company details          |
| Medical Condition  | Disease or health condition        |
| Billing Amount     | Cost of treatment                  |
| Admission Date     | Date of hospital admission         |
| Discharge Date     | Date of discharge                  |
| Doctor Name        | Assigned physician                 |
| Room Number        | Hospital room information          |
